Contenido: Cadenas

¿Qué es una Cadena?

Las cadenas de JavaScript son para almacenar y manipular texto.


¿Entre qué signos deben escribirse las cadenas?

Las cadenas deben escribirse entre comillas.


¿Qué tipo de comillas se pueden utilizar para una cadena?

Con comillas simples, ejemplo:

let carName2 = 'Volvo XC60';

Con comillas dobles, ejemplo:

let carName1 = "Volvo XC60";


¿Se puede usar comillas dentro de una cadena?

Se puede usar comillas dentro de una cadena, siempre que no coincidan con las comillas que rodean la cadena.


¿Con qué propiedad se encuentra la longitud?

Para encontrar la longitud de una cadena, use la length propiedad incorporada.


¿Con qué secuencias se pueden insertar comillas en unacadenas?

La secuencia \' inserta una comilla simple en una cadena.

La secuencia \" inserta una comilla doble en una cadena.

¿Qué es un Objeto?

Los objetos de JavaScript son contenedores para valores con nombre llamados propiedades.


Diferencia entre Objeto y Variable

Las variables de JavaScript son contenedores para valores de datos. Los Objetos también son variables. Pero los objetos pueden contener muchos valores.


¿Cómo y con que se crea un objeto en JS?

Un objeto de JavaScript se crea con un objeto literal. Se crea con const. Ejemplo: const person={firstName:"John", lastName:"Dae", age:"50", eyeColor:"blue"}.

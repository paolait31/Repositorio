Contenido: Bucles

¿Qué es un Bucle?

Los bucles pueden ejecutar un bloque de código varias veces.


¿Cuáles son los diferentes tipos de bucles que JavaScript admite?

for- recorre un bloque de código varias veces.
for/in- recorre las propiedades de un objeto.
for/of- recorre los valores de un objeto iterable.
while- recorre un bloque de código mientras una condición específica es verdadera.
do/while- también recorre un bloque de código mientras una condición específica es verdadera.


Explica las declaraciones del bucle For

La declaración 1 se ejecuta (una vez) antes de la ejecución del bloque de código. Normalmente usará la instrucción 1 para inicializar la variable utilizada en el bucle.
La declaración 2 define la condición para ejecutar el bloque de código. A menudo, la declaración 2 se usa para evaluar la condición de la variable inicial.
La declaración 3 se ejecuta (todas las veces) después de que se haya ejecutado el bloque de código. A menudo, la instrucción 3 incrementa el valor de la variable inicial.


¿Qué recorre la declaración for in?

La declaración de JavaScript for in recorre las propiedades de un objeto.


¿Qué recorre la declaración for of?

La declaración de JavaScript for of recorre los valores de un objeto iterable. Le permite recorrer estructuras de datos iterables como Arrays, Strings, Maps, NodeLists y más.


¿Qué es Variable?

Para cada iteración, el valor de la siguiente propiedad se asigna a la variable. La variable se puede declarar con const, let o var.


¿Qué es Iterable?

Un objeto que tiene propiedades iterables.


¿Qué recorre el ciclo while?

El while ciclo recorre un bloque de código siempre que una condición específica sea verdadera.


¿Qué ejecuta el ciclo do while?

El do while bucle es una variante del bucle while. Este ciclo ejecutará el bloque de código una vez, antes de verificar si la condición es verdadera, luego repetirá el ciclo mientras la condición sea verdadera.
